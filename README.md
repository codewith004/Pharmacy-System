# 💊 Pharmacy Management System

A modern, web-based Pharmacy Management System designed to streamline inventory tracking, billing, and supplier management. Built with vanilla JavaScript and modern web technologies.

## ✨ Features

- **📦 Inventory Management:** Add, edit, and delete medicines with stock tracking.
- **📉 Low Stock Alerts:** Automatic highlighting of medicines with low quantity (< 10).
- **📅 Expiry Tracking:** Visual warnings for expired medicines.
- **💰 Billing & Invoicing:** Generate invoices with automatic total calculation.
- **🖨️ Print Support:** One-click print button for invoices.
- **📊 Dashboard Analytics:** Visual charts showing stock levels.
- **📷 Barcode Scanner:** Integrate camera scanning for quick product lookup (QuaggaJS).
- **👥 Supplier Management:** Track supplier details and contact information.
- **💾 Local Storage:** Data persists even after refreshing the page.

## 🛠️ Technologies Used

- **HTML5** - Structure
- **CSS3** - Styling (Flexbox, Grid, Animations)
- **JavaScript (ES6)** - Logic & DOM Manipulation
- **Chart.js** - Data Visualization
- **QuaggaJS** - Barcode Scanning Library
- **FontAwesome** - Icons
